# Changelog

## v1.2.0

### Added
- Screen-share audio toggle so hosts can include phone/system audio in stream.
- Crop controls for top, bottom, left, and right using percentages.
- Crop aspect-ratio presets (Free, 16:9, 9:16, 4:3, 1:1).
- Localhost hint in Web Access card to simplify same-device browser testing.

### Fixed
- Quick-connect URL now uses the signaling port constant.
- Release script now derives tag from `pubspec.yaml` version to reduce manual mistakes.
